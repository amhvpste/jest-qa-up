Матчеры из React Testing Library

Помимо toBeInTheDocument React Testing Library расширяет стандартный набор матчеров Jest следующими методами:


  • toBeDisabled

  • toBeEnabled

  • toBeEmpty

  • toBeEmptyDOMElement

  • toBeInTheDocument

  • toBeInvalid

  • toBeRequired

  • toBeValid

  • toBeVisible

  • toContainElement

  • toContainHTML

  • toHaveAttribute

  • toHaveClass

  • toHaveFocus

  • toHaveFormValues

  • toHaveStyle

  • toHaveTextContent

  • toHaveValue

  • toHaveDisplayValue

  • toBeChecked

  • toBePartiallyChecked

  • toHaveDescription

Описание и примеры использования можно найти на официальной документации библиотеки https://github.com/testing-library/jest-dom?tab=readme-ov-file#custom-matchers
